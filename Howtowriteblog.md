Blog Article Outline Prompt
Act as an expert SEO strategist tasked with creating a comprehensive blog article outline on the topic of “[Topic / Keyword]”.

Create a numbered, multi-level outline with main topics and subtopics. Follow typical frameworks for titling topics, such as “What is…,” “Benefits of…,” “How to…,” etc. Add a recommended word count for each topic, placed after its title. 

If logically necessary, include relevant, numbered subtopics. For example:

- Include subtopics when a section involves multiple steps (e.g., “How to…”), categories (e.g., “Types of…”), or distinct aspects (e.g., pros and cons, tools, tips).

- Do not include subtopics for straightforward explanatory sections such as “What is…,” “Why use…,” or single-concept overviews.

Keep all topic and subtopic titles short, clear, and easy to understand—avoid jargon, overly complex wording, or unnecessarily long phrases.

List “Introduction” as the first topic of the outline without any subtopics and with a word count of 50 words. Include a "Frequently Asked Questions" section as the last topic in the outline, listing 5–10 highly relevant questions as subtopics.

Ensure that the topics throughout the outline are selected and structured according to the following audience:

[Audience]

Blog Article Meta Description Prompt

Follow these specific instructions for each element:

SEO Title (Max 60 characters)

- Generate 3 title options to choose from
- Include the main keyword near the beginning
- Make the title clear, compelling, and benefit-driven
- Use numbers, brackets, or power words to increase CTR if applicable
- Use title case for improved readability

Meta Description (Max 155 characters)

- Generate 3 meta description options to choose from
- Clearly summarize the article’s core benefit or value
- Naturally include primary and secondary keywords
- Spark curiosity or communicate value
- Use action-oriented language that encourages clicks (e.g., "Learn how...", "Discover...")
- Make it emotionally appealing and informative without overpromising
- Avoid fluff, passive voice, or generic phrases

Tags (Max 10–15 tags)

- Generate a selection of relevant tags based on the article's topic, related subtopics, and semantic keywords
- Include variations of primary and secondary keywords
- Ensure all tags are specific, relevant, and helpful for categorization and discoverability
- Do not use overly broad or generic tags
- Separate the tags with commas
