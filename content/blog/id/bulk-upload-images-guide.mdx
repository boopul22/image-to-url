## Cara Unggah Gambar Secara Massal: Panduan Langkah demi Langkah

### Langkah 1: Akses Alat Unggah Massal

Kunjungi situs web [ImageToURL.cloud](http://www.imagetourl.cloud), dan cukup klik tombol "Klik untuk mengunggah atau tarik dan lepas" di situs web.

### Langkah 2: Pilih Gambar Anda

Di bagian unggah, Anda dapat menarik dan melepas gambar atau cukup klik area unggah. Anda dapat:

- Klik beberapa file dengan Ctrl/Cmd + klik
- Tarik folder lengkap dari komputer Anda
- Posting gambar dalam bentuk file ZIP
- Salin dan tempel dari clipboard

### Langkah 3: Konfigurasikan Pengaturan Unggah

Pilih preferensi Anda:

- Durasi tautan (1 hari, 7 hari, 30 hari atau permanen)
- Pengaturan privasi (publik atau pribadi)
- Kualitas gambar (asli atau optimal)
- Wilayah CDN (otomatis atau situs khusus)

### Langkah 4: Proses dan Hasilkan URL

Klik Unggah, dan foto Anda akan diproses secara bersamaan. Alat ini akan:

- Melakukan optimalisasi gambar untuk mencapai performa web yang baik
- Membuat URL yang berbeda untuk setiap gambar
- Membuat cuplikan kode HTML/Markdown
- Menyediakan tautan langsung dan kode sematan

### Langkah 5: Salin dan Gunakan URL Anda

Setelah pemrosesan, Anda akan memiliki:

- Berbagi URL dari masing-masing gambar
- Opsi ekspor massal (CSV, JSON)
- Kode sematan untuk situs web
- Kode QR untuk berbagi seluler
